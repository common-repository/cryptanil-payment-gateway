Cryptanil Payment Gateway – Its flexible crypto payment gateway for Woocomerce. We support BTC, ETH, USDT, USDC, SOL, TON, TRX, BUSD, BNB and many more.

== Description ==
 
Our Crypto currency Payment Gateway was built with the core values of Crypto currency in mind with regards to anonymity and security. Via plugin your customers can make payment via BTC, ETH, USDT, USDC, SOL, TON, TRX, BUSD, BNB and many more.

The plugin was created to solve a solution that we and several merchants were facing, which was a gateway that respects customer privacy, no external redirects, seamless integration.

### Plugin Features:

Technical Conditions for Using the Plugin 
-	The user should have a website created via Wordpress
-	Woocomerce platform should be inserted into the website
-	The website should comply with the minimum requirements of the plugin 
-	The inner links of the website should be inserted correctly. 

Settings of the Plugin
1.	You can change "Title" and "Description" fields.
3.	Enable/Disable Test mode
4.	Add Test Cryptanil Key
5.	Add Live Cryptanil Key

== Installation == 
1. Install the plugin into your WordPress website. 
2. Register your account with https://www.cryptanil.com 
3. Sign in after sucessfull registration and navigate to https://dashboard.cryptanil.com/admin/settings
4. Take your LIVE and TEST API key and put in plugin settings. 
5. Enable the payment gateway in the settings and click save. 
6. Make sure that Cron jobs are enabled

Done!
